## Setup
Project requires [Node.js](https://nodejs.org/) v14+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ npm install
$ gulp watch
```

For production environments...
```sh
$ the same as above but last commend run
$ gulp
```
